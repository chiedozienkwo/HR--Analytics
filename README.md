# HR Analytics

## Project Overview

This project analyzes **5,020 employee records** using SQL Server and Microsoft Power BI to uncover workforce trends, employee performance, and organizational insights that support data-driven HR decision-making.



## Business Problem

Human Resource departments require reliable workforce insights to improve employee performance, monitor workforce demographics, support promotion decisions, and optimize organizational planning.

The objective of this project was to analyze employee data and develop an interactive dashboard that enables HR managers to monitor key workforce metrics and make informed decisions.



## Tools Used

- SQL Server
- Microsoft Power BI
- Microsoft Excel



## Data Cleaning Process

Before analysis, the dataset was cleaned and validated by:

- Checking for missing values
- Identifying duplicate records
- Verifying data consistency
- Validating employee information
- Preparing the dataset for SQL analysis



## SQL Analysis

SQL Server was used to analyze:

- Employee demographics
- Department performance
- Gender distribution
- Salary analysis
- Employee performance ratings
- Promotion analysis
- Employee attrition
- Workforce trends



 Power BI Dashboard

An interactive Power BI dashboard was created to visualize:

- Employee KPIs
- Department analysis
- Gender distribution
- Salary insights
- Performance ratings
- Promotion analysis
- Attrition insights

The dashboard enables HR managers to quickly monitor workforce performance and identify opportunities for improvement.



## Key Insights

- Analyzed **5,020 employee records**.
- Identified workforce distribution across departments.
- Evaluated employee performance ratings.
- Analyzed salary distribution and promotion trends.
- Examined employee attrition patterns.
- Developed interactive dashboards for HR reporting.



## Business Recommendations

- Strengthen employee retention strategies.
- Monitor departments with high attrition.
- Improve employee development programs.
- Use performance data to support promotion decisions.
- Continue using interactive dashboards to improve workforce planning and decision-making.
